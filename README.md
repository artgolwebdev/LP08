# LP08 - Beauty Salon Landing Page

A luxury beauty salon landing page built with React, TypeScript, and Tailwind CSS.

## 🛠️ Tech Stack

- React 18 + TypeScript
- Tailwind CSS
- Framer Motion
- Vite
- GitHub Pages

## 📦 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
